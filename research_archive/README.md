# Research Archive

This folder preserves historical research and modeling material that is not part of the deployed Streamlit runtime.

Archived here:

- legacy ML and data-collection code from `src/modeling/`, `src/data_collection/`, and `src/data_processing/`
- research-only processed datasets
- notebooks and ML backend documentation
- extra raw datasets that are not needed by the deployed app

The deployed app continues to use:

- `dashboard/`
- `src/services/`
- `src/utils/`
- `src/llm_integration/thumbnail_generator.py`
- `data/youtube api data/`
- `data/assistant/`

This archive is kept for reference and future research reuse, but it is not part of the runtime import surface or deployment contract.
